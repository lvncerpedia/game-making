世界的に「インディーゲームの歴史を変えた」ってレベルで評価されてる作品、実は共通点があるんだよね。

* 小規模チーム or 個人制作
* “好き”を尖らせてる
* AAAがやらない体験をやってる
* 制作者のフェチや思想がそのままゲームになってる

だから開発秘話を知ると、「あ、この人たち異常に執着してるな…」って分かってめちゃ面白い。

「被弾した時、プレイヤーが“自分のせいだ”と思える操作性」を目指したらしい。 ([Wikipedia][1])

“個人の感性が強すぎるゲーム”

# 共通してること

実は全部、

> 「この感情をゲームにしたい」

から始まってる。

| ゲーム            | コア感情     |
| -------------- | -------- |
| Hollow Knight  | 孤独な探索    |
| Undertale      | 良心への問い   |
| Stardew Valley | 安心して暮らす  |
| Hades          | 死んでも前進   |
| Celeste        | 自分を乗り越える |

で、君の監視ゲームってかなり面白い位置にいて、

* “違和感”
* “監視”
* “孤独”
* “気付き”
* “世界理解”
* “不安”

をゲーム化しようとしてる。
かなりインディーゲーム向きのテーマなんだよね。
AAAだと逆に作りづらいやつ。
