# パブリッシャー別 名作ゲーム一覧

## 凡例

| バッジ  | 意味                                 |
| ------- | ------------------------------------ |
| 🏛 殿堂 | ゲーム史に残る殿堂入りタイトル       |
| ⭐ 名作 | 高い評価を得た代表作                 |
| 👑 伝説 | ジャンル・業界を塗り替えた伝説的作品 |

| 記号     | 意味                       |
| -------- | -------------------------- |
| ✅       | プレイ済み                 |
| 👁       | 見た（実況・プレイ動画等） |
| （空欄） | 未プレイ・未視聴           |
